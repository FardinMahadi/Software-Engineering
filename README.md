# Software Engineering

A personal workspace for mastering software engineering fundamentals — structured around the **2026 agentic engineering paradigm**: understanding systems deeply enough to architect, direct, and correct AI coding agents.

## Purpose

Build practical software engineering skills in a structured sequence — from CS core and architecture through security, DevOps, and multi-agent orchestration — while shipping real projects along the way.

## Learning Roadmap

Structured in stages — from fundamentals toward agentic engineering.

> The shift: coding in 2026 means **orchestrating AI agents**, not writing every line. But to correct an agent, you need to know what is right. That's why fundamentals still come first.

| Stage | Topic                                                            | Status    |
| ----- | ---------------------------------------------------------------- | --------- |
| 1     | Programming basics — JS/TS, OOP, APIs, Python, C++               | ✅ Done   |
| 2     | Software architecture — system design, testing, deployment       | 🟣 Active |
| 3     | Git and GitHub workflows — branching, rebase, PR strategy        | 🟣 Active |
| 4     | Security and privacy — auth, OWASP Top 10, data protection       | 🟣 Active |
| 5     | Containerization — Docker, Docker Compose, isolated environments | 🟡 Next   |
| 6     | Agentic engineering — multi-agent orchestration, AI-assisted dev | 🟡 Next   |

Full roadmap with resources: [`learning-roadmap-topics.md`](./learning-roadmap-topics.md)
