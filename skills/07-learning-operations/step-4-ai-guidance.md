# Step 4: AI guidance contract

### Activity: Tell the AI how to help

Create `learning/ai-guidance.md` and document how the AI IDE should behave in this vault.

Cover these points:

1. Which notes the AI must read before suggesting next work
2. How the AI should update progress after a completed task
3. What the AI should preserve in Obsidian notes
4. How the AI should recommend the next learning step
5. What the AI must never do without explicit approval

Use `learning/ai-guidance.md` as the implementation note for this workflow and keep it aligned with the rest of the skill.
