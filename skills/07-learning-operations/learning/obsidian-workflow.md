# Obsidian Workflow

Parent: [[progress/README|Learning Progress Dashboard]]

## Rules

1. Operational notes use YAML frontmatter.
2. Internal references use wikilinks.
3. Tags and properties support navigation and review.
4. Action items use Tasks-plugin-compatible checkboxes.
5. Learning notes link back to a map-of-content note.

## Folder roles

- `progress/` holds active status, trackers, backlog, and reviews.
- `skills/` holds reusable course-style learning paths.
- Root overview docs hold curriculum and repository context, not daily progress state.
