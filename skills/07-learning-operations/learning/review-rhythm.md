# Review Rhythm

Parent: [[progress/README|Learning Progress Dashboard]]

## Daily

- Check [[progress/current-focus|Current Focus]].
- Finish or reschedule the smallest useful task.
- Capture blockers instead of carrying them mentally.

## Weekly

- Run the prompts in [[progress/weekly-review|Weekly Review]].
- Update the progress table in [[progress/skill-tracker|Skill Tracker]].
- Move future items out of active notes and into [[progress/learning-backlog|Learning Backlog]].

## Rule

If a note does not change the next action, it should probably not be edited during a quick review.
