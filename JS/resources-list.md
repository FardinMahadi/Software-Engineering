# JavaScript Resources

## Official References

- [MDN Web Docs: JavaScript reference](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) - authoritative reference for syntax, objects, methods, and browser APIs.
- [ECMAScript specification](https://tc39.es/ecma262/) - the formal language spec for understanding JavaScript behavior and edge cases.

## Learning and Tutorials

- [MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide) - step-by-step tutorials and concepts like closures, prototypes, and async.
- [The Modern JavaScript Tutorial](https://javascript.info/) - beginner-to-advanced lessons with clear explanations and examples.

## Documentation Tools for JS Code

- [JSDoc](https://jsdoc.app/) - popular tool for generating API docs from comment annotations.
- [ESDoc](https://esdoc.org/) - documentation generator with plugins and coverage support.
- [documentation.js](https://documentation.js.org/) - CLI-based doc generator for modern JavaScript and async code.

## Example-Driven Guides

- [JavaScript documentation examples](https://github.com/) - curated examples of function, class, and module docs to adapt for your own projects.
- [Writing documentation for your JavaScript project](https://www.freecodecamp.org/news/tag/documentation/) - practical guidance on what to document, including examples and gotchas.

## Suggested Starting Stack

- For learning JavaScript: start with MDN JavaScript Guide, then use MDN Reference and the ECMAScript spec when you need details.
- For documenting a JS library: start with JSDoc, then add examples and one good documentation generator if needed.
